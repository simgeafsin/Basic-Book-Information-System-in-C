# Basic-Book-Information-System-in-C
# 📚 Mini Library Management System in C

## 📌 Description

This project is a simple **Library Management System** written in C.
It allows users to manage books using basic programming concepts such as **structs, functions, arrays, and pointers**.

The system is console-based and designed for beginners who want to understand how data structures work in C.

---

## 🚀 Features

* ➕ Add new books
* 📋 List all books
* 🔍 Search for a book by name
* ❌ Exit the program

---

## 🛠️ Technologies Used

* C Programming Language
* Standard Libraries:

  * `stdio.h`
  * `string.h`

---

## 📂 How It Works

The program uses:

* `struct` to store book and writer information
* Functions to organize operations
* Arrays to store multiple books

---

## ▶️ How to Run

1. Compile the code:

```
gcc program.c -o program
```

2. Run the program:

```
./program
```
